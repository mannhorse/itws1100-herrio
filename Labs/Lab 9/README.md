This lab builds a PHP and MySQL web application that allows users to add, display, and delete actors and movies from a database. The application connects to a MySQL database  and uses statements to safely add and remove records. The actors page allows you to enter a first name, last name, and date of birth. The movies page allows you to enter a title and year. Both pages show the records in a table with a delete button on each row. This uses jQuery AJAX to remove the record without reloading the page. A third relations tab uses SQL JOIN across the tables to display which actors appeared in which movies.
