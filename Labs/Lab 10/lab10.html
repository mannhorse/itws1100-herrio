Lab 10: Production Deployment

Deploying to production with SSL and continuous auto-deploy.

About This Lab

In this lab, I deployed my website to a live Azure virtual machine, configured HTTPS using Let's Encrypt, set up per-folder password protection, and implemented continuous deployment via a GitHub webhook so every push to main automatically updates the server.

What Was Accomplished

  • Configured a PHP redirect so visiting herriorpi.eastus.cloudapp.azure.com redirects straight to the homepage
  • Protected Lab 1 and Lab 9 with .htaccess HTTP Basic Authentication
  • Added secure flags to the JSON project listings and lock icons to the projects page
  • Installed a free SSL/TLS certificate via Let's Encrypt (Certbot) with auto-renewal
  • Set up a GitHub webhook with HMAC-SHA256 signature verification for automatic deployment on push to main
