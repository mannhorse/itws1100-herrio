This repository has the files for Lab 5 of Intro to ITWS. The goal of this lab is to use JavaScript to validate and HTML form. Features for this lab include client side form validation which ensures no fields are plank, and a custom alert button that displays the first name, last name, and nickname that was entered. Finally, for the assignment 2 "crimes" were fixed. Crime #1 is "Form labels that aren’t associated to form input fields." To fix this, the "for" attribute was added to all the <label> tags it matches the id of the corresponding input box. Crime #4 is "Not indicating an active form field." This was fixed by adding "background-color: #fee;" to the focus state of the form.
